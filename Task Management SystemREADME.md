# Task Management System (Normalized Schema)

This project implements a normalized relational schema for a Task Managemnet System using MySQL.
It demonstrates database design principles, normalization and practical application in managing projects, tasks and collaboration features.

# Project Overview

The Task Management System is designed to help teams organize projects, assign tasks and collaborate effectively.
Key goals of the schema:
-- Data Integrity : Avoid redundancy by normalizing to 3NF.
-- Scalability : Support multiple users, projects and projects.
-- Collaboration : Enable comments and tagging for better communication.
-- Accountability : Track task status and deadlines.

# Example Workflow:

1. A User creates a Project.
2. Tasks are added under the project and assigned to users.
3. Tasks can be tagged with Tags for categorization.
4. Team members add Comments to tasks for collaboration.
5. Task status and deadlines are tracked for accountability.
